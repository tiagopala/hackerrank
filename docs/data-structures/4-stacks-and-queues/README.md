# Stacks & Queues

## Naming

**Stacks**

- C# => Stack<Type>

**Queues**

- C# => Queue<Type>

## Commons Operations

**Stacks**

- `Push` => To put elements on the top of the stack
- `Pop`  => To remove the top element of the stack

**Queues**

- `Enqueue` => To put an element in the end of the queue
- `Dequeue` => To remove an element in the beginning of the queue
